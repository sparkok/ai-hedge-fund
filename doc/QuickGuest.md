# 快速指南：AI Hedge Fund

## 这个程序是做什么的？

输入一组股票代码（如 AAPL、MSFT、NVDA），系统模拟一个**由 AI 分析师组成投资基金**，对每只股票输出交易决策。

每个"分析师"是一个扮演著名投资人的 LLM 智能体——沃伦·巴菲特、查理·芒格、彼得·林奇等——他们读取真实的财务数据（股价、财务指标、新闻、内幕交易），输出一个**信号**：

| 信号 | 含义 |
|------|------|
| bullish | 看涨 —— 认为会涨，考虑买入 |
| bearish | 看跌 —— 认为会跌，考虑卖出/做空 |
| neutral | 中性 —— 无明显倾向 |

然后 **风险管理器** 检查仓位限制和波动率。最后 **投资组合经理** 综合所有信号，决定：买入、卖出、做空、平仓、还是持有——以及多少股。

结果就是具体的**交易预测**。

---

## 支持哪些市场？

| 市场 | 支持？ | 说明 |
|------|--------|------|
| 美股（NYSE/NASDAQ） | ✅ 支持 | AAPL、MSFT、NVDA 等标准美股代码 |
| 中国 A 股（上海/深圳） | ❌ 不支持 | 数据源 financialdatasets.ai 仅接入 SEC（美国证监会）注册公司 |
| 港股 | ❌ 不支持 | 同上 |
| 其他国际市场 | ❌ 不支持 | 仅限美股 |

**数据源：** 这个项目使用 [financialdatasets.ai](https://financialdatasets.ai) 获取股价、财务指标、新闻等数据。该 API 仅覆盖美国上市公司（SEC 注册）。所以这个程序**只能分析美股**，无法直接用于 A 股或港股。

> 如果你想分析 A 股，需要替换 `src/tools/api.py` 中的数据源为一个支持 A 股的 API（如 Tushare、AkShare、Wind 等），并调整数据模型。这属于较深度的二次开发。




## 工作流程

```
输入股票代码（如 AAPL, MSFT）
       │
       ▼
┌───────────────────┐
│   获取财务数据     │  ← 从 financialdatasets.ai 获取
│  （股价/新闻/      │     （需要 FINANCIAL_DATASETS_API_KEY）
│   内幕交易等）     │
└───────┬───────────┘
        │
        ▼
┌──────────────────────────────┐
│   AI 分析师（你选择）         │  ← 各自读取数据，输出信号
│   - 沃伦·巴菲特              │
│   - 查理·芒格                │
│   - 彼得·林奇                │
│   - 技术分析师               │
│   - 情绪分析师               │
│   - ...最多 19 个            │
└──────────┬───────────────────┘
           │
           ▼
┌──────────────────────────────┐
│   风险管理器                  │  ← 检查波动率、仓位上限
└──────────┬───────────────────┘
           │
           ▼
┌──────────────────────────────┐
│   投资组合经理 (LLM)          │  ← 综合信号+风险，
│                              │     给出每只股票的最终操作
└──────────┬───────────────────┘
           │
           ▼
    买入 / 卖出 / 做空 / 平仓 / 持有
    （附带股数 + 推理理由）
```

## 快速上手（命令行）

### 前提

```bash
# 1. 安装依赖（一次）
poetry install

# 2. 设置 API Key
cp .env.example .env
# 编辑 .env —— 至少需要：
#   OPENAI_API_KEY=sk-...
#   FINANCIAL_DATASETS_API_KEY=...
```

### 预测：默认股票

```bash
poetry run python src/main.py --ticker AAPL,MSFT,NVDA
```

程序会引导你：
1. **选择分析师** —— 空格选中/取消，回车确认。每个名字旁有中文翻译。
2. **选择模型** —— 选择 LLM 供应商（GPT-4、Claude 等）。

选择确认后，浏览器会自动打开 Bing 搜索每位分析师，方便你了解他们是谁。

### 预测：指定时间段

```bash
poetry run python src/main.py --ticker AAPL --start-date 2024-01-01 --end-date 2024-06-01
```

### 使用本地模型（Ollama）

```bash
poetry run python src/main.py --ticker AAPL,MSFT --ollama
```

### 回测（历史模拟）

```bash
poetry run python src/backtester.py --ticker AAPL,MSFT --start-date 2024-01-01 --end-date 2024-12-31
```

## 如何理解输出

终端输出示例：

```
Analysis for AAPL
══════════════════════════════════════════

Agent                 信号       置信度
───────────────────────────────────────────
Warren Buffett        BULLISH     0.85
Charlie Munger        NEUTRAL     0.60
Technical Analyst     BULLISH     0.72
Risk Management       —           —       ← 仓位上限: $10,000

Portfolio Decision for AAPL
────────────────────────────
操作        买入
数量        10 股 (~$1,800)
理由        多个看涨信号，置信度较高，且在风控范围内
```

分析师信号中各字段含义：

| 字段 | 含义 |
|------|------|
| `signal` | bullish / bearish / neutral — 分析师的预测方向 |
| `confidence` | 0.0–1.0 — 分析师的确信程度 |
| `reasoning` | 简短文字说明判断逻辑 |

最后的 **Portfolio Decision（投资组合决策）** 就是系统的最终预测。

## 使用建议

- **分析师不是越多越好。** 3–5 个风格不同的分析师（如巴菲特 + 技术分析师 + 情绪分析师）往往比 19 个全选信号更清晰。
- **先用回测验证。** 在信任单次预测之前，先用 backtest 在历史数据上跑一遍。
- **只支持美股。** 不能用 A 股代码（如 600519）或港股代码。
- **数据来源：** 所有财务数据来自 [financialdatasets.ai](https://financialdatasets.ai)。免费版有调用次数限制。
- **本地模型：** 如果不想付费调用云端 API，用 `--ollama` 搭配本地模型如 `llama3` 或 `qwen2.5`。
